## 1. Введение

Расширение `loan_annuity` предназначено для расчёта аннуитетных платежей по кредиту, построения графика погашения, вычисления полной стоимости кредита (ПСК) и экспорта графика в формат JSON.

Расширение разработано для PostgreSQL 16 и использует язык PL/Python3. Оно может быть полезно для финансовых приложений, банковских систем, а также для учебных целей при изучении кредитных расчётов.

## 2. Состав расширения

Расширение включает следующие объекты базы данных:

### 2.1. Тип данных `payment_row`

Составной тип, описывающий одну строку графика платежей:

| Поле | Тип | Описание |
|------|-----|----------|
| `payment_date` | date | Дата платежа |
| `mea` | numeric | Минимальный ежемесячный платёж (МЕП) |
| `interest_amount` | numeric | Сумма процентов в МЕП |
| `principal_amount` | numeric | Сумма основного долга в МЕП |
| `remaining_balance` | numeric | Остаток долга после платежа |
| `interest_rate` | numeric | Годовая процентная ставка |

### 2.2. Таблица `payment_schedules`

Хранит все сгенерированные графики платежей:

| Поле | Тип | Описание |
|------|-----|----------|
| `accountid` | integer | Идентификатор договора |
| `calcid` | integer | Номер расчёта (в рамках одного договора) |
| `calc_date` | date | Дата построения графика |
| `payment_date` | date | Дата платежа |
| `mea` | numeric | МЕП |
| `interest_amount` | numeric | Сумма процентов |
| `principal_amount` | numeric | Сумма основного долга |
| `remaining_balance` | numeric | Остаток после платежа |
| `interest_rate` | numeric | Процентная ставка |

Первичный ключ: `(accountid, calcid, payment_date)`

### 2.3. Функции

#### `build_schedule(accountid integer, amount numeric, rate numeric, months integer, issue_date text)`

Строит график аннуитетных платежей, вставляет его в таблицу `payment_schedules` и возвращает результат в виде набора строк типа `payment_row`.

**Параметры:**
- `accountid` – идентификатор договора
- `amount` – сумма кредита
- `rate` – годовая процентная ставка (в %)
- `months` – срок кредита в месяцах
- `issue_date` – дата выдачи кредита (в формате YYYY-MM-DD)

**Возвращает:** `SETOF payment_row`

#### `calculate_psc(accountid integer, calcid integer)`

Вычисляет полную стоимость кредита (ПСК) по сохранённому графику.

**Параметры:**
- `accountid` – идентификатор договора
- `calcid` – номер расчёта

**Возвращает:** `numeric` – ПСК в процентах (например, 7.35)

#### `export_schedule_json(accountid integer, calcid integer)`

Экспортирует график платежей в формате JSON.

**Параметры:**
- `accountid` – идентификатор договора
- `calcid` – номер расчёта

**Возвращает:** `jsonb` – график в виде JSON-массива

## 3. Математические формулы

### 3.1. Аннуитетный платёж

Ежемесячный аннуитетный платёж рассчитывается по формуле:

$$
P = S \times \frac{i \times (1 + i)^n}{(1 + i)^n - 1}
$$

где:
- $P$ – ежемесячный платёж
- $S$ – сумма кредита
- $i$ – месячная процентная ставка, $i = \frac{\text{годовая ставка}}{12 \times 100}$
- $n$ – срок кредита в месяцах

### 3.2. Полная стоимость кредита (ПСК)**

ПСК рассчитывается численным методом (метод Ньютона) из уравнения:

$$
S = \sum_{t=1}^{n} \frac{P_t}{(1 + \text{ПСК})^{t/12}}
$$

где:
- $S$ — сумма кредита
- $P_t$ — платёж в периоде $t$
- $n$ — количество платежей
- $t$ — номер платежа (в месяцах)

## 4. Установка расширения

### 4.1. Требования

- PostgreSQL 16
- PL/Python3 (пакет `postgresql-plpython3-16`)

### 4.2. Установка PL/Python3 (если не установлен)

```bash
sudo apt update
sudo apt install postgresql-plpython3-16
sudo systemctl restart postgresql
```

### 4.3. Установка расширения

1. Перейдите в каталог с исходными файлами расширения:
```bash
cd /loan_annuity
```

2. Выполните установку через Makefile:
```bash
sudo make install
```

3. Подключитесь к базе данных и создайте расширение:
```sql
CREATE EXTENSION loan_annuity;
```

## 5. Примеры использования

### Пример 1. Построение графика платежей для кредита 1 000 000 ₽ на 12 месяцев под 12% годовых

```sql
SELECT * FROM build_schedule(
    1,          -- accountid
    1000000,    -- сумма
    12,         -- ставка (%)
    12,         -- срок (мес)
    '2024-01-01' -- дата выдачи
);
```
**Результат:** таблица из 12 строк с датами платежей и суммами.

<img src="./assets/2026-03-28 162050.jpg" width="700">

### Пример 2. Расчёт ПСК для этого же кредита

```sql
SELECT calculate_psc(1, 1);
```
**Результат:** `12.68` (процентов)

<img src="./assets/2026-03-28 162148.jpg" width="500">

### 5.3. Экспорт графика в JSON

```sql
SELECT jsonb_pretty(export_schedule_json(1, 1));
```

<img src="./assets/2026-03-28 164823.jpg" width="400">

**Результат:** JSON-массив с полной информацией о каждом платеже.
