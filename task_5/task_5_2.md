## Упражение 2

### Задание
Этот запрос выбирает из таблицы «Билеты» (`tickets`) всех пассажиров с именами, состоящими из трех букв (в шаблоне присутствуют три символа `«_»`):
```sql
SELECT passenger_name
FROM tickets
WHERE passenger_name LIKE '___ %';
```
Предложите шаблон поиска в операторе `LIKE` для выбора из этой таблицы всех
пассажиров с фамилиями, состоящими из пяти букв.

### Решение

Для выбора всех пассажиров с фамилиями, состоящими из пяти букв, можно использовать следующий шаблон в операторе LIKE:
```sql
SELECT passenger_name
FROM tickets
WHERE passenger_name LIKE '% _____';
```
Объяснение шаблона '% _____':

- % - любое количество любых символов (для имени)

- пробел (разделитель между именем и фамилией)

- _____ - пять символов _ (каждый _ означает один любой символ)

`LIMIT 5` установлен для краткости вывода результата на экран.
